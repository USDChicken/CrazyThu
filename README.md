## Installation

Before running this project, you need to install its dependencies. Run the following command:

```shell
npm install
```


To run the sample deployment script:

```shell
npx hardhat accounts
npx hardhat compile
npx hardhat clean
npx hardhat test
npx hardhat node
node scripts/sample-script.js
npx hardhat help
```
